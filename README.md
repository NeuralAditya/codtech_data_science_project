# 🚀 CodTech Data Science Internship Projects

This repository contains all the tasks completed as part of the **CodTech Data Science Internship**.
The projects demonstrate practical implementation of **data pipelines, deep learning, full-stack deployment, and optimization techniques** using Python.

---

## 📌 Overview

The internship required completing **4 major data science tasks**, covering the full lifecycle of data-driven systems:

* Data preprocessing & ETL pipelines
* Deep learning model development
* End-to-end deployment of ML models
* Optimization using mathematical programming

---

## 🧠 Tasks Implemented

---

### 🔹 Task 1: Data Pipeline Development

**Objective:**
Build an automated pipeline for data preprocessing, transformation, and loading.

**Tools Used:**

* pandas
* scikit-learn

**Key Features:**

* Data loading from CSV
* Handling missing values
* Feature scaling using StandardScaler
* Exporting processed dataset

📁 Folder: `task1-data-pipeline/`

---

### 🔹 Task 2: Deep Learning Project

**Objective:**
Implement a deep learning model for image classification.

**Tools Used:**

* TensorFlow / Keras

**Key Features:**

* MNIST dataset classification
* Neural network model
* Training and evaluation
* Accuracy visualization

📁 Folder: `task2-deep-learning/`

---

### 🔹 Task 3: End-to-End Data Science Project

**Objective:**
Develop and deploy a machine learning model using a web framework.

**Tools Used:**

* Flask
* scikit-learn
* NumPy

**Key Features:**

* Model training and serialization (`model.pkl`)
* REST API for predictions
* JSON-based input/output
* Ready for cloud deployment

📁 Folder: `task3-full-project/`

---

### 🔹 Task 4: Optimization Model

**Objective:**
Solve a business problem using linear programming.

**Tools Used:**

* PuLP

**Key Features:**

* Profit maximization problem
* Constraint-based optimization
* Mathematical modeling
* Solution interpretation

📁 Folder: `task4-optimization/`

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/codtech-data-science-internship.git
cd codtech-data-science-internship
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Projects

### Task 1:

```bash
python pipeline.py
```

### Task 2:

Open and run the notebook in Jupyter/Colab.

### Task 3 (Flask App):

```bash
python app.py
```

Then open:

```
http://127.0.0.1:5000/
```

### Task 4:

Run the notebook or Python script.
